# Stab

A Chrome extension that automatically closes idle, duplicate, and memory-heavy tabs to keep your browser clean and fast.

## Features

- **Idle Tab Closing** - Automatically close tabs that haven't been active for a configurable time (default: 30 minutes)
- **Duplicate Detection** - Close duplicate URLs, keeping only the most recently active tab
- **Memory Hog Detection** - Optionally close tabs using excessive memory (requires Chrome 116+)

## Local Installation

1. Clone this repository
2. Open Chrome and go to `chrome://extensions`
3. Enable "Developer mode" (toggle in top right)
4. Click "Load unpacked" and select the `stab` folder
5. Click the extension icon to configure settings
